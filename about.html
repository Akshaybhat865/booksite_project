Book site

The online book review project is to implement a computer-based online book review. This system objective is to the system design Book Review, and to develop a customer web-based Review Management system. This website provides the book reader to register them self’s on the website and give review of the books read by them. The book review website serves as a platform for various book readers to give their experience of reading for the other reading who are in search of a good read. The website is equipped with various book which are already stored in the database and also provides a provision to add a book in case it is not stored on the website.

A website that helps a user to read and write the reviews of book and explore some wonderful books around the world. Website gives a monthly subscription for a user read the books. A user friendly website that makes the work easy foe the users.
